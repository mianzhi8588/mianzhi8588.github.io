## Overview

This project contributes to a Harvard Innovation Lab startup by developing a cloud-based backend system for quotation and pricing workflows.

## My Contributions

- Built modular Python components for product-size calculation and pricing-related logic.
- Developed FastAPI-based backend APIs.
- Integrated user authentication, file management, and database-backed result storage.
- Deployed backend services on AWS EC2.
- Used Git and GitHub for version control and collaboration.

## Technical Stack

- Python
- FastAPI
- AWS EC2
- Git / GitHub
- Database-backed storage
- File management
