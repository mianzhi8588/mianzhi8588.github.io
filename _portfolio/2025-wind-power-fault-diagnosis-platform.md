## Overview

This project focuses on intelligent fault diagnosis for wind power generation systems using multimodal SCADA data and deep-learning models.

## My Contributions

- Built a deep-learning algorithm library for abnormal-state detection and fault diagnosis.
- Implemented models including CNN, CNN-LSTM, RNN, One-Class SVM, and Normalizing Flow.
- Worked on class imbalance, cross-device generalization, and online-inference constraints.
- Explored source-domain-free deployment strategies for industrial wind-farm settings.

## Technical Stack

- Python
- PyTorch
- SCADA data analysis
- CNN / CNN-LSTM / RNN
- One-Class SVM
- Normalizing Flow
