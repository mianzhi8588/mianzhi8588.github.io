## Overview

This project develops a desktop software system for bearing fault diagnosis, converting research-stage scripts into a more usable tool for data preprocessing, model training, intelligent diagnosis, and visual analysis.

## My Contributions

- Developed a PyQt5 desktop application with integrated modules for data preprocessing, model training, diagnosis, and visualization.
- Implemented FFT, time-domain and frequency-domain feature extraction.
- Integrated CNN-LSTM diagnostic models with interpretable outputs.
- Validated the system on benchmark bearing datasets.

## Technical Stack

- Python
- PyQt5
- PyTorch
- FFT
- Time-frequency analysis
- CNN-LSTM
